# D3.Q3.3 Experiment - Mojibake Scan

## Experiment E3.3

Question: Where do UTF-8 mojibake markers remain across runtime source, content JSON, and docs?

Method:

- Reuse the D3 audit scanner.
- Scan:
  - `lib/**/*.dart`
  - `assets/data/content/**/*.json`
  - `docs/**/*.md`
- Flag common mojibake marker substrings:
  - `Ã`
  - `Â`
  - `â€`
  - `â†`
  - `â‰`
  - `âœ`
  - `â”`
  - `�`
- Group hit counts by root and file.

Command:

```text
dart run tool\research\vietnamese_i18n_audit_report.dart --app-language lib\core\app_language.dart --lib-root lib --content-root assets\data\content --docs-root docs
```

Note: the scan intentionally avoids raw `â` because it is a valid Vietnamese letter.
