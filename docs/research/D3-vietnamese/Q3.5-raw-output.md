## Existing Flutter Localization Hook

`pubspec.yaml` includes:

```yaml
dependencies:
  flutter_localizations:
    sdk: flutter
```

`lib/app/app.dart` already imports `flutter_localizations` and configures:

```dart
locale: language.locale,
supportedLocales: supportedAppLocales,
localizationsDelegates: GlobalMaterialLocalizations.delegates,
```

`lib/core/language_provider.dart` stores explicit app language choice in SharedPreferences key `app.locale` and defaults to Vietnamese only when the device locale language code is `vi`; otherwise English.
