# D3.Q3.1 Experiment - App Language Audit

## Experiment E3.1

Question: Is `lib/core/app_language.dart` structurally complete enough to trust as the central UI-copy source?

Method:

- Add a reproducible D3 audit scanner.
- Count localized `return` statements by `AppLanguage.en`, `AppLanguage.vi`, and `AppLanguage.ja`.
- Count blank/TODO/draft localized returns.
- Record file length and return-count balance.

Command:

```text
dart run tool\research\vietnamese_i18n_audit_report.dart --app-language lib\core\app_language.dart --lib-root lib --content-root assets\data\content --docs-root docs
```

Initial observation: the file has `7,561` lines and `2,040` `case AppLanguage.*` matches, suggesting about `680` localized members before exact parsing.
