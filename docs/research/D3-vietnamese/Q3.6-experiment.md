# D3.Q3.6 Experiment - Plural and ICU Risk

## Cheapest Experiment

Search actual Dart source for:

1. Existing ICU/plural APIs.
2. Manual singular/plural guards.
3. English interpolated count strings with plural nouns.
4. Central `app_language.dart` count-label helpers that should be fixed first.

## Commands

```text
rg -n "Intl\.plural|plural" lib -g "*.dart"
rg -n '== 1 \? '''' : ''s''|== 1 \? "" : "s"|count == 1|activeCount == 1|completedCount == 1|length == 1' lib -g '*.dart'
rg -n '\$count (items|terms|lessons|questions|strokes|due)|\$[a-zA-Z]+Count (items|terms|lessons|questions|decks)|\$\{[^}]+\} (items|terms|lessons|questions|decks|prompts)' lib -g '*.dart'
rg -n "String unitMinutesLabel|String unitMinutesApprox|String itemsCountLabel|String lessonCountLabel|String unansweredSubmitLabel|String handwritingStrokeShortLabel|String termsNeedPracticeLabel|String stillInReviewQueueLabel|String reviewTermsDueLabel|String kanaDueTodayLabel" lib\core\app_language.dart
(rg -n "\$\{[^}]+\}|\$[A-Za-z_][A-Za-z0-9_]*" lib -g "*.dart" | Measure-Object).Count
```

## Caveat

Regex search overcounts code interpolation and undercounts multiline/template-generated copy. It is a triage pass, not a full ICU migration manifest.
