# D3.Q3.2 Experiment - Hardcoded Vietnamese Outside AppLanguage

## Experiment E3.2

Question: How much Vietnamese UI copy bypasses `lib/core/app_language.dart`?

Method:

- Reuse the D3 audit scanner.
- Scan `.dart` files under `lib/`, excluding `lib/core/app_language.dart`.
- Flag lines containing Vietnamese Quốc Ngữ diacritics.
- Group hit counts by file.

Command:

```text
dart run tool\research\vietnamese_i18n_audit_report.dart --app-language lib\core\app_language.dart --lib-root lib --content-root assets\data\content --docs-root docs
```

Manual seed observation: `rg` found `358` Vietnamese-diacritic matches across `27` non-`app_language.dart` Dart files.
