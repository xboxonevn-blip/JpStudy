# D3.Q3.4 Experiment - Vietnamese Typography Quality

## Cheapest Experiment

Build a fixed-seed sampler over Vietnamese UI strings, then score `100` strings with simple, reproducible typography heuristics before any manual rewrite.

## Candidate Set

- `lib/core/app_language.dart`: Vietnamese localized return strings.
- `lib/**/*.dart` outside `app_language.dart`: hardcoded Dart string literals containing Vietnamese diacritics.
- Exclude `lib/core/research/` so audit code and regex definitions do not become learner-facing samples.

## Scoring

Each sampled string starts at `5`.

- `question_mark_mojibake`: `-3`
- `raw_english_term`: `-2`
- `ascii_ellipsis`: `-1`
- `ascii_dash`: `-1`
- `space_before_punctuation`: `-1`
- `tone_variant_review`: `-1`

The score floor is `1`.

## Command

```text
dart run tool\research\vietnamese_typography_audit_report.dart --app-language lib\core\app_language.dart --lib-root lib --sample-size 100 --seed 20260514
```

## Caveat

This is not a full human editorial review. It detects cheap objective typography/terminology warnings and creates a reproducible sample for follow-up manual review.
