## What This Means

- Mojibake is no longer a broad content-JSON or Dart-source blocker under the current marker set.
- Non-UTF8 docs were a tooling risk; the audit now has a regression guard that catches future decode failures.
- Encoding checks should become a small CI/report gate before broad editorial work.

## Red Team

The marker set is intentionally conservative. It avoids raw `â` because that is a valid Vietnamese character, so it can miss some mojibake variants. Conversely, it may miss non-Vietnamese encoding corruption that does not use these markers.
