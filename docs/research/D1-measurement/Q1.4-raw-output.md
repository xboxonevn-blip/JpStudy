Relevant observed schema facts:

```text
Dataset name: analytics_<property_id>

Intraday table: events_intraday_YYYYMMDD
event_timestamp: microseconds UTC
event_name: event name
event_params: repeated RECORD with key and typed value fields
user_id: app-assigned user id
user_pseudo_id: pseudonymous app-instance/user id
```
