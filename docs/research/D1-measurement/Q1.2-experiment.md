# D1.Q1.2 Experiment - Synthetic User Replay

## Experiment E1.6

Question: Can we run one command that spawns 10 deterministic simulated users with different personas and returns an NS report?

Method:

- Add `SyntheticNorthStarEventSimulator.generate(...)` that emits NS event families for persona-tagged users.
- Add CLI flag `--simulate-users <n> --window-start <iso>` that converts generated events through the same `NorthStarEventMapper` used by exports.
- Verify determinism, N5-N1 persona coverage, and CLI scoring through tests.

Expected information gain: high. This exercises the event-level replay path without waiting for real GA4 export access.

Cost: 0.5-1 hour.

Seed: `jpstudy-phase0-ns-v1`

Command:

```text
dart run tool\research\north_star_report.dart --simulate-users 10 --window-start 2026-05-01T00:00:00.000Z
```
