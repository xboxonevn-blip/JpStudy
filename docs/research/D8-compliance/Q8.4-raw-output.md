## External Documentation Checks

Firebase Crashlytics Flutter docs say setup requires adding `firebase_crashlytics`, running `flutterfire configure`, configuring `FlutterError.onError`, configuring `PlatformDispatcher.instance.onError` for async errors, and forcing a test exception before relying on the dashboard.

Sentry Flutter docs say the Flutter SDK can automatically report errors and performance data, can be configured through `SentryFlutter.init`, and supports Dart defines for `SENTRY_DSN`, release, dist, and environment. Sentry troubleshooting calls out web zone setup and web source maps.
