# D4.Q4.P2 Analysis — Anh Tuấn N3

Commit: `8387e444`
Date: 2026-05-14

## Result

P2 is not production-ready for a busy N3 learner. The core N3 learning surfaces exist, but route initialization, exam entry, mobile CTA safety, and Vietnamese copy defects create high-friction failures on normal entry paths.

## Hypothesis Updates

H4.P2.1 — Level persistence: falsified. Root cold start applies N3, but direct deep links skip app init and default level-sensitive screens to N5. Root cause in local code: `appInitProvider` is watched inside `HomeScreen`, not globally at app shell startup.

H4.P2.2 — 15-minute planning: partially supported. Desktop home shows `~14 phút` with a clear plan. Mobile home collapses to broad cards and hides the actual 15-minute plan detail.

H4.P2.3 — JLPT readiness: partially supported. Coach, reading, and full mock are reachable after root init. The sidebar exam route/live deploy is inconsistent, quick mock bank is `0Q`, and full mock is 31 minutes, too long for the stated daily habit.

Follow-up 2026-05-14: live `/exam-center` drift is deploy drift, not current local route code. The deployed `main.dart.js` maps both `/exam-center` and `/practice/mock-exam` builders to the same mock exam widget, while current local `exam_routes.dart` maps `/exam-center` to `ExamCenterHubScreen`.

H4.P2.4 — Commute mobile: falsified. The reading CTA can sit under the bottom nav; a tap intended for reading navigated to Kanji. Kanji/vocab also interrupt N3 with a foundations modal.

H4.P2.5 — Vietnamese UI: falsified. Live reading chips show `3 c?u` / `9 ph?t`; mobile vocab leaks `Ready now`. A local TDD patch fixed study-goal labels and JLPT reading meta pill helpers, but live remains pre-fix.

## Severity

- HIGH: Deep links do not initialize persisted level; direct N3 links can silently become N5.
- HIGH: Mobile bottom nav can occlude a primary reading CTA and route the user away.
- HIGH: Exam entry is inconsistent on live; `/#/exam-center` does not match local richer route behavior.
- MEDIUM: N3 vocab page reports `Hajimete N3 (0 mục từ)` despite broader D2 evidence that N3 vocab content exists elsewhere.
- MEDIUM: N3 users get foundations modal interruptions on vocab/kanji.
- MEDIUM: Live Vietnamese mojibake and English leakage remain visible.

## What Worked

- Root-start N3/VI state works when prefs are correctly JSON-encoded.
- Desktop home exposes a plausible 14-minute plan.
- Grammar N3 hub shows correct deck size (`100`) after root init.
- JLPT coach gives a useful N3 baseline frame and reading drill target.
- Reading drill offers 25 N3 passages with a 9-minute target, matching a commute session.

## Red Team

The UAT used one browser session with residual auth/session state, so the sidebar exam stale-route behavior may depend on prior branch stack. However, hard reload still showed live route drift from local code, and direct grammar cold start independently proves a global init problem. The CTA occlusion was observed at one mobile viewport (`390x844`); it needs viewport matrix confirmation before exact layout fix.

## Action

1. Move `appInitProvider` watch out of `HomeScreen` into app/shell bootstrap so every deep link loads persisted level/goal. Status: local patch added in `lib/app/app.dart` with `test/app/app_init_deep_link_test.dart`; live deploy still unverified.
2. Redeploy `jpstudy-v2` hosting or verify the intended channel; live `/exam-center` is confirmed stale against local route code.
3. Add mobile bottom-padding/CTA visibility tests for JLPT reading cards. Status: local patch added a `390x844` regression test and compact mobile reading layout.
4. Decide whether N3 soft foundations gates should appear once, be dismissible permanently, or be disabled for N3+.
5. Patch remaining vocab copy/data route issue: `Ready now`, `Hajimete N3 (0 mục từ)`.

## Mental Model

Một app học theo cấp độ chỉ đáng tin khi cấp độ được nạp trước mọi route. Nếu N3 chỉ đúng khi đi qua trang chủ, còn link thẳng rơi về N5, người học sẽ tưởng nội dung sai hoặc mất thời gian. Với người bận, mỗi lần app hỏi lại nền tảng, che nút chính, hoặc đưa sang route khác là mất một phiên học 15 phút.
