# D7.Q7.6 Hypotheses - Font Subset Verification

## H7.6.1 - Material Icons are tree-shaken

Prediction: release `MaterialIcons-Regular.otf` is below `100 KB` raw.

Falsifier: Material Icons remains near full upstream size or exceeds `300 KB`.

## H7.6.2 - Cupertino Icons are negligible

Prediction: release `CupertinoIcons.ttf` is below `10 KB` raw.

Falsifier: Cupertino icons become a measurable first-route payload.

## H7.6.3 - Product font is the largest local font

Prediction: bundled Manrope is larger than icon fonts but below `200 KB` raw and below `70 KB` Brotli.

Falsifier: Manrope dominates transfer size or is not compressed by Hosting.

## H7.6.4 - Live first route does not fetch unexpected remote fonts

Prediction: the first route fetches only bundled local fonts.

Falsifier: `fonts.gstatic.com` font requests appear during startup.

## H7.6.5 - Font payload is not the current D7 blocker

Prediction: total live font transfer is much smaller than `main.dart.js`, CanvasKit, and content/runtime assets.

Falsifier: font transfer approaches app shell or renderer transfer sizes.
