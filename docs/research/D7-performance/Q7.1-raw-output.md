## Caveats

- Playwright navigation timing on localhost is not a Lighthouse score and is not field telemetry.
- Python `http.server` does not apply Firebase Hosting headers, compression, CDN caching, or live referer policy.
- Browser session had a local security extension injecting requests; ignore Kaspersky `gc.kis...` calls for product performance.
- Initial page fetched many grammar JSON assets; this is a stronger product risk than localhost FCP because real mobile networks will pay the request/transfer cost.

## E7.3 Lazy Grammar Seed Remediation

Root-cause trace:

```text
databaseProvider -> Future.microtask -> GrammarSeeder.seedGrammarData()
GrammarSeeder.seedGrammarData() -> loads N5/N4/N3/N2/N1 grammar JSON

ContentDatabase.onCreate/beforeOpen -> _seed/_ensureMinnaGrammar()
_seed/_ensureMinnaGrammar() -> loads all grammar definition/example JSON
```

RED evidence:

```text
flutter test test\data\seeds\grammar_seeder_lazy_seed_test.dart --concurrency=1 --reporter=expanded
Expected: equals ['N5'] unordered
Actual: ['N1', 'N2', 'N3', 'N4', 'N5']

flutter test test\data\repositories\lesson_repository_test.dart --concurrency=1 --reporter=expanded
Expected: non-empty
Actual: []
```

Patch:

```text
GrammarSeeder.seedGrammarData() now seeds only SharedPreferences onboarding.level.
GrammarSeeder.seedGrammarDataForLevel(level) seeds a requested level on demand.
ContentDatabase.ensureGrammarSeededForLevel(level) seeds requested content grammar on demand.
LessonRepository.seedGrammarIfEmpty() filters by lessonId + jlptLevel and ensures requested content level first.
JLPT mock grammar section ensures its requested level before querying content DB.
```
