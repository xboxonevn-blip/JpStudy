# D7.Q7.2 Hypotheses - Bundle Composition

## H7.2.1 - Dart app JS is not the only large payload

Prediction: `main.dart.js` is large, but Flutter renderer/runtime payloads and static content assets explain most raw build bytes.

Falsifier: top-level app JS accounts for more than half of `build/web`.

## H7.2.2 - Content JSON remains the largest app-owned payload family

Prediction: app-owned content/support JSON is larger than images, fonts, and generated manifests combined.

Falsifier: images/fonts/manifests exceed content/support JSON by raw bytes.

## H7.2.3 - Route code is not currently split into deferred chunks

Prediction: the release build emits a single `main.dart.js` app bundle without route-level deferred JS chunks.

Falsifier: build output contains route/deferred `part` JS files.

## H7.2.4 - Source maps can identify source families but not exact byte ownership without extra tooling

Prediction: a `--source-maps` build can count app/package/sdk source families, but precise byte attribution still needs a source-map size explorer.

Falsifier: generated source map directly reports per-source byte sizes.
