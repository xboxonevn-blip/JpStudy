# D6.Q6.1 Experiment - Card Pattern Audit

## Experiment E6.1

Question: Are card surfaces consistent enough for a mixed-module beta, or do learners see different visual grammars across home, vocab, kanji, grammar, exam, and learn flows?

Method:

- Read shared UI primitives and theme tokens.
- Grep core feature modules for `AppSectionCard`, `AppFeatureCard`, `Card`, `BoxDecoration`, `borderRadius`, `boxShadow`, and `HomeSurface.softPanel`.
- Sample 20 representative card/surface widgets across home, vocab, kanji, grammar, exam, test, and learn.
- Compare border radius, padding, shadow/elevation, header pattern, and CTA placement.
