# D2.Q2.1 Analysis - Vietnamese Content Review Status

## E2.1 Result

Current content review metadata is not production-ready for a 100-learner beta. The strongest explicit debt is N1/N2 grammar examples: `1,744` items are both `machine-translated-vi` and explicitly open for review; another `142` grammar explanations carry `manual-review-needed`.

## Findings

1. Upper-level risk is concentrated. N1 has `4,701 / 8,389` machine-origin items and N2 has `2,752 / 4,770`.
2. Open review is concentrated. `1,744 / 1,886` explicit open-review items are in `grammar_examples`; the remaining `142` are grammar explanations tagged `manual-review-needed`.
3. Vocabulary is status-ambiguous. `vocab` has `5,273` machine-origin items but `0` open-review and `0` approval signals, so machine-origin vocab cannot be interpreted as reviewed or unreviewed from tags alone.
4. Grammar explanations show a different state. `grammar` has `436` machine-origin items and `436` approval signals, meaning the machine-origin tag was retained after approval.
5. N3-N5 have `142` open-review grammar signals and `0` approval signals in this scanner. That does not prove the rest are reviewed; it proves current lower-level files still have incomplete status metadata.
6. March 16 audits are stale for full-scope readiness. They scanned `351` files; current JSON scope is `781`, a `122.5%` increase.

## Red Team

Steelman opposite: explicit tags may undercount real human review because older N3-N5 content predates status metadata. That is plausible, but it strengthens the need for a status taxonomy before claiming production readiness. Absence of review debt tags is not evidence of learner-safe Vietnamese, and Q2.2 sampling already found one alias (`manual-review-needed`) that the first pass missed.

## Decision

Do not start broad content rewriting from grep hits. First normalize the review taxonomy so every content item can answer:

- source: human-authored, imported, machine-assisted, or derived;
- Vietnamese review state: unreviewed, needs edit, approved, or rejected;
- reviewer/date/source notes where available.

## Action

D2.2 should define and backfill a minimal review-status schema, then rank edit work by learner exposure:

1. N5/N4 first-screen paths and SRS cards.
2. N1/N2 grammar examples with explicit open review.
3. N1/N2 vocab machine-origin rows with missing approval/open-review status.
