# D2.Q2.2 Experiment - Grammar Explanation Sample Quality

## Experiment E2.2

Question: What does a reproducible 10-item grammar-explanation quality sample reveal beyond tag counts?

Method:

- Draw 2 grammar explanations per JLPT level from `assets/data/content/grammar`.
- Use deterministic SHA-256 ordering with seed `jpstudy-d2-q2.2-v1`.
- Rate each item manually on:
  - Vietnamese fluency: 1-5
  - Pedagogic clarity: 1-5
  - Accuracy vs source fields: 1-5
  - "Would I be confused as a learner?": Y/N
- Treat `titleEn`, `structure`, and `explanationEn` as the local source for this cheap pass. This is not a full external grammar-source verification.

Sampling command:

```text
$seed='jpstudy-d2-q2.2-v1'; enumerate assets/data/content/grammar/**/*.json; hash "$seed|$relativePath|$index"; sort by hash; take first 2 per level.
```

Output file required by the objective:

```text
docs/research/D2-content-sample-eval.md
```
