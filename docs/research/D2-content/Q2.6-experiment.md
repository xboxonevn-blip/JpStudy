# D2.Q2.6 Experiment - Kanji Han-Viet / Unihan Spot Check

## Experiment E2.6

Question: Are current N3/N2/N1 kanji Han-Viet labels complete and consistent with Unicode Unihan `kVietnamese`?

Method:

- Add a reproducible local spot-check scanner:
  - `lib/core/research/kanji_unihan_spot_check.dart`
  - `tool/research/kanji_unihan_spot_check_report.dart`
- Use TDD:
  - `test/core/research/kanji_unihan_spot_check_test.dart`
  - `test/tool/research/kanji_unihan_spot_check_report_test.dart`
- Compare local `labels.hanViet` with Unihan `kVietnamese`.
- Normalize only case and whitespace; keep accents significant.
- Deterministically sample by stable hash over `seed|level|kanjiId|character`.

Command:

```text
dart run tool\research\kanji_unihan_spot_check_report.dart --content-root assets\data\content --unihan-readings .codex\sources\Unihan\Unihan_Readings.txt --sample-size 50 --seed jpstudy-d2-q2.6-v1
```
