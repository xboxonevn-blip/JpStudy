# D2.Q2.4 Experiment - Cross-Link Coverage Graph

## Experiment E2.4

Question: Are vocab, kanji, and grammar content items cross-linked enough to support prerequisite-aware learning?

Method:

- Add a reproducible content graph scanner:
  - `lib/core/research/content_link_graph.dart`
  - `tool/research/content_link_graph_report.dart`
- Use TDD:
  - `test/core/research/content_link_graph_test.dart`
  - `test/tool/research/content_link_graph_report_test.dart`
- Scan local content only.

Metrics:

- Vocab to kanji:
  - vocab entries with kanji;
  - entries whose kanji are fully covered by same-level kanji dataset;
  - unique vocab kanji vs kanji dataset characters.
- Kanji examples:
  - example words found in same-level vocab terms;
  - `sourceVocabId` refs found in same-level vocab ids;
  - supports both `lemma.vocabId` and `links.sourceVocabId`.
- Grammar examples:
  - example groups whose `grammarPoint` normalizes to a local grammar `title`;
  - total example sentences.

Command:

```text
dart run tool\research\content_link_graph_report.dart --content-root assets\data\content
```
