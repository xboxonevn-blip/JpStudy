# D2.Q2.5 Hypotheses - Content Scope vs JLPT Rough Scope

## H2.5.1 - Vocab volume is not the main upper-level gap

Prediction: cumulative local vocab counts meet rough JLPT volume targets through N1, because N3-N1 have multiple imported routes. Effect size: cumulative local vocab is >= 90% of rough target at each level.

Falsifier: cumulative local vocab falls below 70% of rough target for N2 or N1.

## H2.5.2 - Kanji volume is the main upper-level scope gap

Prediction: cumulative local kanji counts are near enough for N4/N5 but materially below rough N2/N1 expectations. Effect size: cumulative N1 kanji is < 60% of a 2,000-kanji rough target.

Falsifier: cumulative N1 kanji is >= 80% of rough target.

## H2.5.3 - Example sentence volume is uneven but not the first blocker

Prediction: grammar example sentence counts are adequate in lower levels and thinner in N1/N2, but the clearer blocker is content quality/link coverage rather than raw example count. Effect size: each level has hundreds of grammar examples.

Falsifier: any level has fewer than 100 grammar example sentences.
