# D2.Q2.5 Experiment - Content Scope vs JLPT Rough Scope

## Experiment E2.5

Question: For each JLPT level N5 -> N1, how much distinct local content exists, and where are scope gaps?

Method:

- Add a reproducible local scope scanner:
  - `lib/core/research/content_scope_report.dart`
  - `tool/research/content_scope_report.dart`
- Use TDD:
  - `test/core/research/content_scope_report_test.dart`
  - `test/tool/research/content_scope_report_test.dart`
- Count by level:
  - distinct vocab terms from `lemma.term`;
  - distinct kanji from `kanji[].character`;
  - distinct grammar point titles;
  - grammar example sentences.
- Compare to rough JLPT counts only after noting that the official JLPT no longer publishes vocabulary/kanji/grammar lists.

Command:

```text
dart run tool\research\content_scope_report.dart --content-root assets\data\content
```

Reference sources:

- Official JLPT FAQ, no published item lists: https://www.jlpt.jp/e/faq/
- JLPT official guidebook, old/new test specs note: https://www.jlpt.jp/e/reference/pdf/guidebook_s_e.pdf
- Kanjidon rough level counts: https://kanjidon.com/fr/blog/combien-kanji-jlpt/
- JLPT Hub N2 rough counts: https://www.jlpthub.com/jlpt-n2-kanji-practice-guide-with-stroke-order-and-examples/
- JLPTBooks N1 rough counts: https://www.jlptbooks.com/jlpt-n1-complete-guide/

Reference caveat: rough counts are not official current JLPT requirements.
